# Nicolás Capetillo — Press Kit Digital

Press kit / portfolio personal del actor mexicano Nicolás Capetillo. Sitio estático construido con Astro, desplegado en Vercel.

## Comandos

| Comando | Acción |
|---------|--------|
| `npm install` | Instalar dependencias |
| `npm run dev` | Servidor local (`localhost:4321`) |
| `npm run build` | Build producción a `dist/` |
| `npm run preview` | Preview del build |
| `npx astro check` | Verificar tipos |

## Contenido

Todo el contenido del sitio se encuentra en `src/data/nicolas.json`. Las imágenes van en `public/assets/`.

## Despliegue

Automático vía GitHub + Vercel. Cada push a `main` dispara build y deploy.
